# side-scrolling-java
Trabalho prático desenvolvido para a disciplina de Linguagem e Técnicas de programação II do curso de Informática pelo CEFET-MG. Consiste na recriação de um jogo do gênero side-scrolling utilizando a linguagem Java e a biblioteca gráfica LibGDX. O jogo implementado é uma adaptação da primeira fase do título "Castlevania", da Nintendo (NES).
